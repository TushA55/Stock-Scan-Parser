## Project Structure

This project follows the Clean Architecture principles and is separated in the following modules:

```
lib/
  config/
    routes/
    theme/
  core/
    constants/
    resources/
    usecases/
    util/
  features/
    dashboard/
      data/
        data_sources/
        models/
        repository/
      domain/
        entities/
        repository/
        usecases/
      presentation/
        bloc/
        pages/
        widgets/
injection_container.dart
main.dart
```

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/TushA55/Stock-Scan-Parser.git
```

### 2. Install the dependencies

```bash
flutter pub get
```

### 3. Run the app

```bash
flutter run
```

## Running the tests

```bash
flutter test
```
